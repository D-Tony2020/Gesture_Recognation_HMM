## 字段说明

- **输入特征**: 使用了哪些 IMU 通道（如 "6D全部"、"仅陀螺仪3D"、"归一化后6D"）
- **M (簇数)**: k-means 的 n_clusters 参数
- **训练数据**: 用于拟合 k-means 的数据来源（如 "train set 1 全量"、"train set 1+2 合并"）
- **评估指标**: silhouette score、inertia、或下游 HMM 分类准确率
- **备注**: 特殊处理（如去除时间戳列、特征标准化、异常值过滤等）

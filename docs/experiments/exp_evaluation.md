## 字段说明

- **模型组合**: 使用的 6 个手势 HMM 模型 ID 集合（如 "H001–H006"）
- **数据集**: val (自己划分的验证集) / train-additional (单次手势训练集) / test (最终测试集)
- **Top-1 准确率**: 最高 LL 模型即为预测，正确率
- **Top-3 准确率**: 前 3 个 LL 最高的模型中包含正确答案的比例
- **主要混淆模式**: 哪些手势对容易互相混淆（如 "Eight ↔ Infinity"）
